1 кейс: Сохранения и загрузки данных при отсутствии файла

Цель: убедиться, что программа правильно создает и загружает данные при отсутствии файла
1. Запустить программу
2. Создать новую задачу
3. Сохранить
4. Выйти из программы
5. Запустить ее снова
6. В списке задач должна отображаться созданная вами задача
Ожидаемый результат:
- задача есть в списке.


2 кейс : Поиск по ключевому слову

Цель: убедиться в правильности работы поиска 
1. Запустить программу
2. Создать две задачи
3. Выполнить поиск по слову, которое имеется в одной задаче, но отсутвует в другой
4. Проверить, что отображается нужная задача
Ожидаемый результат:
- в результатах поиска отображается нужная задача 
- вторая задача не отображается
